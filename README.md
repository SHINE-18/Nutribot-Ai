# NutriBot 🍏
An AI-powered assistant that helps users evaluate the nutritional value of packaged food and decide if it's worth buying.

## 🔍 Features
- Barcode/Nutrition Label Scanner (manual input for prototype)
- Health scoring system (Rule-based logic)
- Smart food recommendations (coming soon)
- Chat-like interface via Streamlit

## 🚀 How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run app: `streamlit run app.py`

## 🧠 Tech Stack
- Python, Streamlit
- OCR (Tesseract - for future enhancement)
- Food info database: OpenFoodFacts (planned)
- LangGraph/LLM APIs (planned)
