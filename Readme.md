
# 🍎 NutriBot-Ai — Your Personal AI Food Assistant

Hello everyone! 👋  
Welcome to **NutriBot-Ai**, a project I built during the **AI GTU IBM SkillsBuild Summer Internship 2025**.

## 🎯 Goal

To help people make **smarter, healthier food choices** using personalized AI analysis — offering nutrition scores, smart suggestions, and a chat-based interface.

## 🚀 Features

- 🔐 Secure Sign-in (Convex Auth) with Anonymous Mode
- 🧠 Personalized food analysis powered by **OpenAI GPT**
- 📊 Health score (1–10) & smart food alternatives
- 💬 Real-time AI chat assistant
- 🧾 Profile customization (goals, allergies, conditions)
- 📈 Food history log with previous queries

## 📦 Getting Started

### Prerequisites:
- Node.js v16+
- npm

### Installation:
```bash
git clone https://github.com/SHINE-18/Nutribot-Ai.git
cd Nutribot-Ai
npm install
npx convex dev   # Backend
npm run dev      # Frontend

Set up your environment:

OPENAI_API_KEY=your_openai_key
CONVEX_DEPLOYMENT=your_convex_project_url
```
## Usage Guide
1. Sign in (or use anonymous mode).

2. Set up your health profile: age, weight, goals, allergies.

3. Ask queries like:
- “Is Diet Coke healthy?”
- “Suggest something better than Maggi.”
- “Compare Milk vs Almond Milk.”
  
🎯 All responses are logged for future tracking.
